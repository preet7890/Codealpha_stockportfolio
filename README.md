# Codealpha_stockportfolio
Stock Portfolio Tracker

Description:

This is a simple Stock Portfolio Tracker developed using Python.
The program allows users to enter stock names and quantities to calculate the total investment value.

Features:

- Stores stock prices using dictionary
- Calculates investment value
- Accepts multiple stock entries
- Displays total portfolio value
- Console-based application

Technologies Used:

- Python

Concepts Used:

- Dictionary
- Loops
- Conditional statements
- Variables
- User input/output

How to Run:

1. Save the file as "stock_tracker.py"
2. Open it in Python or Pydroid 3
3. Run the program

Example:

User enters stock name and quantity, and the program calculates the total investment amount.
